By default, Odoo updates partner data when opening the client list. With the module it's not necessary to make an extra action, because it allows to update partner data in POSes instantly after a partner data has been modified.

How It Works

Open POS session


Imagine that another user (e.g. administrator) change the customer data at this time, e.g. barcode


As a result, in opened POS the partner data is updated instantly without any extra action and new barcode is valid now.


Configuration

You are able to select the fields to be synced in the Point of Sale → Configuration → Settings menu
