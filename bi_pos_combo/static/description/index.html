Odoo POS-Point of Sales Combo Products Apps

This odoo apps is used to offer several products for POS sale as one combined product in point of sales in Odoo as app, which also helps to combined multiple product pack in single product with grouped of product price .This Odoo module helps to sell one single grouped product in single price which includes several products item in one bundle pack product.

Features

Configure Combo Product

User can configure combo product in POS and Product.

Create POS order with Combo Product

Create POS order with required and optional products in combo products.

Can alter optional product from cart in POS

User can alter the created cart to alter the optional product in combo product.

Create Multiple Combo Product, Combo on Order, Payment & Receipt

User can create multiple combo products and generate POS receipt with combo products.

Calculate Combo Price from Main Product.

Calculate combo price from only main product of pos combo.

Calculate Combo Price from All Combo Products.

Calculate combo price from all of products added in pos combo.

POS Combo Configuration

Under Point of Sale > Configuration > Point of Sale user need to enable "Use combo in POS", user can also select option for 'Total Combo Price' as "Total of all combo items" or "Take Price from the Main product".

Combo Product Configuration

Under product user need to enable "Is Combo Product" option to use product as combo, User can see "Combo Products" tab and add products from selected category for combo, user can also make product as "Required" in combo.

User can also see price of added products to combo.

POS Combo Product

User can see point of sale combo product as shown image on point of sale screen.

Option 1: Total of all combo items

On clicking combo item on pos screen new pop up will open, user can see all added "Required Products" and "Optional Products" user can select optional product as per choice and click "Apply" button.

Combo Price

User can see price for combo product calculated from total of selected combo items.

POS Receipt with Combo Product

User can see point of sale order receipt with combo product and price.

POS Order with Combo Product

User can see create point of sale order with combo product.

Option 2: Take Price from the Main product

Combo Product Price Configuration

User have to select "Take Price from the Main product" option to calculate combo item price from main product.

User can also see price of main combo product.

On clicking combo item on pos screen new pop up will open, user can see all added "Required Products" and "Optional Products" user can select optional product as per choice and click "Apply" button.

Combo Price

User can see price for combo product calculated from the main combo product.

POS Receipt with Combo Product

User can see point of sale order receipt with added combo item and price.

POS Order with Combo Product

User can see point of sale order with selected combo product.

Alter Combo Product

Click on the button shown in image on POS cart to alter pos combo optional product.

From "POS Combo Products" pop-up user can change optional products as per choice.

Updated Combo Products

User can see updated combo products for pos combo.
