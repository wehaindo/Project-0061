Odoo POS Show Discounted Price

Odoo POS Show Discounted Price Module enables you to create a price list, attach the same price list to the point of sale and show both discounted price as well as the actual price of the product in Point of Sale. This will enable you to show the difference between the actual price and discounted price.

Show amazing discounts and make customers feel special!!

If you are planning to have some special discounts on products in your Point Of Sale and want to show the actual price along with the discounted price, then this module is very useful to you.

Now provide amazing discounts and showcase them in your store. This makes customers happy and satisfied that compels them for future purchasing too.

Using this module along with the Point Of Sale module, we can create a price list, attach the same price list to the Point Of Sale and show both discounted price as well as the actual price of the product in Point Of Sale. In this way, the cashier has the knowledge that the product's price has been deducted and he/she can tell the customer as well.

Detailed Features List

Below is the detailed list of Feature for Odoo POS Show Discounted Price

Create price list from the backend in Odoo

  • The Odoo POS Show Discounted Price module allows you to create a price list from the back-end.
  • The price list shows both 'Actual' as well as 'Discounted' price of products in your POS.

View the prices in the running POS Session itself

  • The cashier can know both the prices in the running POS session itself.
  • He/She does not need to check the back-end for the prices

Set discount in POS on the products accordingly

  • The Discount can be set in POS on all products from the backend.
  • Admin can even set the discount in POS specifically for some products also.

Show the final discounted prices to your customers too

  • Product's actual and discounted price are visible for customers too, on the receipt.
  • This makes customers happy and satisfied that compels them for future purchasing too.
