Odoo 14 POS Customer Wallet(Community)

This module allows user to add, change to the wallet, redeem it and print Wallet Ledger.

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

How to use

  • Enable wallet configurations.


  • When the customer is selected, buttons will get appeared as shown below.


  • Customer can add change in wallet.


  • Wallet amount display in receipt.


  • Clicking on "Add To Wallet" button will show popup to add amount.


  • Wallet amount display in receipt..


  • Check this box to automatically add change to wallet.


  • As shown below customer's wallet amount can be seen and you can also use "Add to Wallet" feature from there.


  • Customer's wallet history will be displayed as shown below.


  • Click on "Wallet Ledger" to open popup as shown below.


  • Ledger can be printed in the receipt as shown below.


  • User can view records in the "Wallet" menu.
