WEHA Smart POS - POS Order Backup Restore

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Modification and Features

  • Show or hide delete unpaid order on pos (same function with delete unpaid order on debug mode)
  • Show or hide delete paid order on pos (same function with delete paid order on debug mode)
  • Show or hide export paid order on pos (same function with export unpaid order on debug mode)
  • Show or hide export unpaid order on pos (same function with export paid order on debug mode)
  • Show or hide import order to current pos session (same function with import order on debug mode)
