POS Cash Register End Of Session Z-Report Odoo Apps

POS Cash register End Of Session Z-Report Odoo Apps support both community as well as enterprise version. This Point of sales module will allow user to print POS Current and Closed Session Reports which consist total tranasaction and sales summary report at End of shift and total sales summary of session. You can generate report for Current and Closed sessions in PDF format and This odoo apps is POS-BOX Compatible.

Features

POS Configuration for Enable Current Sesion Report

Under point of sale configuration user can tick on checkbox for enable current session report.

Print Current Session Report from POS Screen

User can print only current session report directly from POS screen in PDF format.

Print Session Report from POS Session

User can print and view session wise report from POS session in PDF format.

Posted Session Report

Using Posted Session Report, user can generate report for Closed sessions.
